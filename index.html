EVOLUTION AI PROFESSIONAL


The First Easy to Use & State of the Art Evolution Big Data Artificial Intelligence Desktop Software

EvolutionAI is an evolution big data Artificial Intelligence learning/training & prediction application. The word 'Evolution' indicates that the AI model trained by the software can be trained further with new knowledge/data and this without the need for having the data learned in the past. EvolutionAI is also optimized for handling big data. You can feed data files to EvolutionAI with any size, even several GB's at once, and this for both training/learning and prediction.

EvolutionAI Professional is an Artificial Intelligence (AI) software toolkit and may be used for many different purposes as business process improvement, decision making, making decisions about buying/selling stocks on the stock exchange, knowledge management, illness and cure discovery, R&D in all disciplines and sectors, etc. The possible applications are endless.

The Main Advantages

These are a few of the advantages of EvolutionAI compared to other currently existing  AI solutions:

  1. - Can be trained further with new knowledge/data. There is no need for retraining the AI model with old data.
     
  2. - Adjustable learning capacity.
     
  3. - Optimized for big data. There is no size limit for training and prediction.
     
  4. - Desktop MS Windows (64-bit) application which means that your data stays in your company (there is no need for internet). If you have sensitive data (e.g. personal records, company records, financial data, etc.) then EvolutionAI is the definitive solution without the need for sending your data over the internet!
     
  5. - Built-in easy to use big data cleaning pre-processing. You can easily automate the cleaning of your data of any size. Cleaning data for AI applications was a lot of work before but EvolutionAI makes this a fast and simple semi-automatic process.
     
  6. - Built-in automatic big data statistical analyzes. Not only segments of your big data are analyzed but the whole data set of any size!
     
  7. - Automatic Classification of unclassified data. The data you are importing will be automatically classified (grouped) by the AI according to often invisible trends in the data.
     
  8. - Very fast! Several millions of data records are processed in a few minutes.
     
  9. - Automatic report in PDF or Word format.
     
  10. - Built-in server which can be used to automatically process data files for prediction with a trained AI model. You can easily connect EvolutionAI with any software package being used in your company (ERP, CRM, etc.) by using the EvolutionAI server!

Load Data

You can import/load new data files with the Load Data button. The Load Data module will guide you through two steps to clean your data and to transfer and select any desired columns and rows. There is an easy to use built-in filter query editor which can be used to automatically select the necessary data (per column and row) depending on simple expressions as for example the sum of the values in several columns, the value in a specific column, etc.

You can choose from several delimiters which will be used to split the data rows into several columns of data (variables). Many types of delimited text data files are supported. Even MS Excel files can be imported.

The Load Data module will also automatically classify your data if you check this option. This can be very useful in case it would be too much work to classify the data yourself or if it is not obvious how to classify the data. The AI will search for hidden patterns in your data and classify it accordingly.

Analyze Data

The Analyze Data module is automatically launched at the end of loading a data file and it can also be opened later to view the analyzes results of any data file in the project. The analyzes shows all important statistical properties of each data column, the distribution of the data in each column and also the correlation between the data columns. After training the AI the analyzes results may also include a recommendation from the AI with information on how to improve your data. The automatic report module will make a PDF or MS Word report from the analyzes with a click of a button.

Not only segments of your big data are analyzed but the whole data set of any size!


View & Edit External Data

With the built-in BigFileEditor you can view and edit external (not registered in the project) data files in text format with up to 2GB size (please note that this is only a size limit for the external text editor but not for EvoultionAI!). BigFileEditor can also be used to search and replace any text in the file easily. BigFileEditor is very fast and optimized for viewing and editing very large files which you can not even open with e.g. Notepad.

Some of the available operations in BigFileEditor
  • Search and Replace (Search menu).
  • Set Bookmarks to jump through very large chunk of data to an exact location (Search menu).
  • View line numbers (View menu).
  • Convert Line End characters (Options menu) e.g. if you have a data file from another operating system.
  • Change indentation/tab distances (Options menu). This can be very interesting to have all of your delimited data columns at the same location for easy viewing.
In case your external text file is larger than 2GB then you first need to split it into several parts with the built-in huge file splitter. Data files which you import into an EvolutionAI project do not need to be split first (there is no size limit)!

Training The AI Model

Only data registered in the EvolutionAI project can be used for training an AI model. All types of delimited text files (with any delimiter) and also MS Excel files can be loaded into an EvolutionAI Project. The Load Data module (see above) will guide you through two steps to clean your data and to transfer and select (filter) any desired columns and rows.

You can save the trained AI model into the project and use it for prediction or train it further with new data any time you want. The AI will report the accuracy of the AI model each time it is trained. The accuracy of the EvolutionAI model is most of the time above 99% when well designed data is used for training. The strategy (which variables to use, the amount of each decision class) and  the quality of the data are very important for a good AI model!

The word 'Evolution' indicates that the AI model trained by the software can be trained further with new knowledge/data and this without the need for having the data learned in the past.

With the automatic parameter optimization module you can automatically determine the optimal combination of the parameters influencing the AI model.

Prediction

After training an EvolutionAI model the model can be used for prediction of the classification of a data record (one or several rows of data).

You can use a data file loaded into the project with the Load Data command or you can enter the data into the Prediction Form. You can also export an MS Excel or delimited text file template and fill in your input data for prediction. The template will have exactly the same data structure than the data used for training the AI model.

The Output Data files with the predictions (first column) and with the probability estimates of the predictions (second column) will be saved in the Project folder.

The EvolutionAI Server

The EvolutionAI server can be used to automatically process data files for prediction with a trained AI model. You can for example easily connect EvolutionAI with any software package being used in your company (ERP, CRM, etc.) by using the EvolutionAI server and automate the prediction process. The EvolutionAI server is extremely fast and it can predict millions of records in just a few seconds.
